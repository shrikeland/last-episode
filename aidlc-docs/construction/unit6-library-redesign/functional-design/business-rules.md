# Business Rules — Unit 6: Library Redesign

## BR-01: Режим отображения

- По умолчанию (нет активных фильтров) — **режим секций**: три горизонтальных ряда, сгруппированных по типу тайтла.
- При активном поиске (`search` не пуст) **или** при выбранном статусе (`status ≠ 'all'`) — **режим грида**: плоская сетка, аналогичная предыдущей реализации.
- Переключение происходит автоматически на основе URL-параметров; отдельной кнопки нет.

## BR-02: Порядок секций

Секции всегда отображаются в фиксированном порядке:
1. 🎬 Фильмы (`type = 'movie'`)
2. 📺 Сериалы (`type = 'tv'`)
3. ⛩️ Аниме (`type = 'anime'`)

Если в секции нет ни одного тайтла — секция скрывается полностью (не рендерится).

## BR-03: Горизонтальная прокрутка

- Ряд прокручивается горизонтально. Системный скроллбар скрыт (`.scrollbar-hide`).
- Кнопки ← / → появляются при наведении курсора на блок секции.
- Левая кнопка ← отображается только если `scrollLeft > 4px` (не у левого края).
- Правая кнопка → отображается только если `scrollLeft + clientWidth < scrollWidth - 4px` (не у правого края).
- При клике на кнопку контейнер прокручивается на `80% своей ширины` в соответствующую сторону с `behavior: 'smooth'`.
- По краям ряда отображаются fade-градиенты (from-background) для визуального намёка на продолжение.

## BR-04: Фильтр по типу удалён

- Дроплист «Тип» убирается из `FilterBar`, так как типы теперь выражены визуальными секциями.
- Параметр `type` в URL более не используется на странице библиотеки.
- Оставшиеся фильтры: поиск, статус, сортировка.

## BR-05: Сортировка применяется внутри каждой секции

- Выбранная сортировка (по дате, названию, рейтингу) действует на уровне серверного запроса — все тайтлы отдаются уже отсортированными.
- Группировка по типу происходит после получения данных, порядок элементов внутри ряда сохраняется.

## BR-06: Счётчик секции

- Рядом с названием секции отображается количество тайтлов в ней.
- Счётчик не интерактивен (не фильтрует).

## BR-07: Пустое состояние

- Если у пользователя нет ни одного тайтла (все секции пусты) — отображается `EmptyState` с предложением добавить первый тайтл.
- В режиме грида (при активных фильтрах) при отсутствии результатов — `EmptyState` с кнопкой «Сбросить фильтры».
